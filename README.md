# 📱 Mobile Sales Analysis | Power BI Dashboard

## 📌 Overview

This project is an end-to-end **Power BI Sales Dashboard** built to analyze mobile phone sales across different cities, brands, payment methods, and time periods. The dashboard transforms raw sales data into meaningful business insights using Power BI, Power Query, and DAX.

---

## 🚀 Project Objectives

* Analyze overall mobile sales performance.
* Compare sales across different brands.
* Track monthly sales trends.
* Analyze city-wise sales distribution.
* Evaluate customer ratings.
* Understand payment method preferences.
* Build an interactive dashboard for business decision-making.

---

## 📊 Dashboard Features

* Total Sales KPI
* Total Quantity KPI
* Total Transactions KPI
* Average Sales KPI
* City-wise Sales Map
* Monthly Sales Trend
* Sales by Mobile Model
* Sales by Day Name
* Customer Rating Analysis
* Transaction Analysis by Payment Method
* Brand-wise Summary Table
* Interactive Slicers

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query
* DAX
* Microsoft Excel

## 📈 Key Insights
